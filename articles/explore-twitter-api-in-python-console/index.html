There are many more commands to explore in this way. All you have to do is read the twitter api documentation and digg out what you need. I have tried to dug up these commands for you to play around. Have Fun!

#   Python-Twitter Library, Twitter Api, Python Console

#   Import twitter library
>>> import twitter

#   Create twitter Instance
>>> api twitter.Api()

#   Create twitter Instance with OAuth
>>> api = twitter.Api(consumer_key='XXXXXXXXXXXXXXXXXXXX', consumer_secret='XXXXXXXXXXXXXX', access_token_key='XXXXXXX-XXXXXXXXXXXXX', access_token_secret='XXXXXXXXXXXXXXXXXXXXXX')

#   Verify your OAuth credentials
>>> print api.VerifyCredentials()

#   Authenticate and get your friends list
>>> users = api.GetFriends()

#   Print User names from the received data
>>> print [u.name for u in users]

#   Print User Ids from the received data
>>> print [u.id for u in users]

#   Get your timeline data
>>> status = api.GetUserTimeline()

#   Print your latest tweets
>>> print [s.text for s in status]

#   Print favorited tweets (True/False)
>>> print [s.favorited for s in status]

#   Print coordinates
>>> print [s.coordinates for s in status]

#   Print meta data
>>> print [s.created_at for s in status]

#   Print Id
>>> print [s.id_str for s in status]

#   Print Retweet counts
>>> print [s.retweet_count for s in status]

#   Print tweets user name
>>> print [s.user.name for s in status]

#   Print tweets user location (If any)
>>> print [s.user.location for s in status]

#   Post a tweet
>>> status = api.PostUpdate('tweeting via python console using python-twitter and twitter api. Its fun! #twitter #api #python #console http://bitben.org')

#   Print status of the posted tweet
>>> print status

#   Get followers data
>>> status = api.GetFollowers()

#   Print names of your followers
>>> print [s.screen_name for s in status]

#   Get Direct Messages
>>> status = api.GetDirectMessages()

#   Print Direct Messages
>>> print [s.text for s in status]

#	Print Sender Screen Name
>>> print [s.sender_screen_name for s in status]

#   Read More about these commands
>>> help(twitter)

Once again Twitter api is not limited to the above commands. It is much more… All you have to do is read twitter api documentation here https://dev.twitter.com/docs . The following were just few examples to show you how exactly to work with twitter api and python twitter. For more information Read python-twitter documentation in python console itself.
