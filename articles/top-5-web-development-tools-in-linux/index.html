Top 5 web development tools in linux

Code Editor : Sublime Text

There are many Linux code editors out there, but I choose sublime text as the best code editor for web development because its sleek, elegant with amazing performance. Even though this software is not officially included in the repository (Ubuntu,fedora or any Linux distro), it can be easily installed by unpacking the zipped file and make the sublime text file executable. That’s it, Now you can double click and start working with the best code editor.

Designing software : GIMP

If you are using windows or Mac OS, there is an awesome image editing tool called Photoshop. It has wide range of features and options for image editing. So far GIMP is the best imaging editing software available in Linux. If you are just beginning for the first time with GIMP you may find it a little annoying but once when you learn the tool placement and its uses, you will never think of going back to photoshop. Even though i began my work life with photoshop now i prefer to use GIMP in Linux because of my annoying feeling towards windows(its bugs).

Browser : Firefox

When it comes to browser there is a war between Google chrome and Firefox, its very hard to say which one is the best but as far as i have experienced during the development process; I feel Firefox is much stable than Google chrome. The reason is Google chrome crashes while using chrome developer tools, firebug or any other apps. Google chrome has been considered as the fastest web browser and it is a fact too, but as i said u will get pissed off when your browser crashes every now and then. Firefox is much more stable as per my experience. Some developers haven’t faced these kind of issues and are satisfied with Google chrome’s performance. Firefox is the best companion for me during my web development project, and may be chrome is yours.

Addons : Web developer tools

While you are working on developing websites or web application, live modification is what you expect without changing a line of code from any files. whether it is validating your code, checking for broken links/images, enabling or disabling scripts, checking responsive behavior of the site and much more. You need a powerful tool in order to accomplish these things. That is why “Web developer tools” has been mentioned as one of the top tools required for web development. Version control and Bugs tracking : GIT

The last important thing is maintaining your precious codes. A small mistake in the code, for example: missing a brace, accidentally deleting a line of code, or overwriting an existing code can make you panic when the application is not working as intended to. This is where you feel a need of version control system and bug tracking. Version control system like GIT can keep track of your code. If you find there is a mistake in the present version of code then you can roll back immediately right to the particular stage where the code was working properly.
