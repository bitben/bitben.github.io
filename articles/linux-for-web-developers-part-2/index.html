Linux for web developers - part 2

In the previous episode we discussed on two operating systems Windows & Linux for Web Developers where we suggested Linux would be the best choice for web developers. In this episode we are going to get started with listing and analysing of top Linux operating systems and essential software’s for a newbie and especially for Web Developers. So lets get started.

We will be categorizing this episode into 3 Bytes:

  • Linux Operating System – First byte
  • Essential Software’s – Second byte
  • Installing software’s to get started – Third byte

Lets have the first byte – Linux Operating Systems

let me give a small introduction on operating systems. There are many operating systems out there but all does the same thing. A program that interacts between the machine hardware and a computer user. Its a simple definition instead of giving a mind attacking one.

Linux Operating System

There are thousands of Linux Operating System out there fulfilling different purposes of computer users. If you are a developer and you need specific operating system for your task, well there is a good news that you can choose out of many or you can customize according to your need. Out of which the major ones that are taking place of Windows and Mac are…

  • Ubuntu
  • Linux Mint
  • Fedora
  • Debian
  • Arch Linux

In which Fedora (RED HAT Based) is my favourite one

All these Operating Systems are good for Web Developers and newbie users can prefer Ubuntu, Linux Mint & Fedora. Even though Fedora targets advanced users, it can full-fill the basic needs of a newbie user with little bit of Linux understanding.

1. Ubuntu

It is a Debian based distribution targeting first time Linux users with easy understanding procedures to install it on a computer. It has unity desktop environment and Ubuntu is supported by canonical. Well if you are a newbie to the Linux field then Ubuntu is your best choice. Wait that doesn’t mean it cannot be used for advanced purposes. Since its a Linux distribution it has all the advanced features for you to work with it. but you need to learn Linux commands for advanced task to be carried out. this is a good distribution for web developers who are stepping into Linux world for the first time.

2. Linux Mint

It is an Ubuntu based distribution targeting first time Linux users with a very good visual interface included. i,e two desktop environments (Mate – Originally forked from legendary GNOME2 & Cinnamon – Originally forked from GNOME3). It is almost similar to Ubuntu distribution with a new look and a self hosted repository containing some additional Debian packages(software’s).

3. Fedora

It is based on legendary RED HAT Linux distribution targeting advanced Linux users. It is also a bleeding edge distribution to some extent. It is a security enhanced Linux distribution which can also be used as a web server. The main thing which I like in fedora is its security features and GUI, It uses GNOME3 Desktop environment which is my favourite.

4. Debian

Debian is one of the oldest Linux distribution which is very stable when compared to other Linux distributions. Debian repository has around 21,000 Debian software’s if i am correct, which works totally fine without any hurdles. Even though it is a stable Linux distribution sometimes as a Web Developer we feel not to use it since most of the Debian Software’s are of older versions. That means the new packages(versions of software’s) are not tested with the stable distribution. So it can be considered as a very good distribution for servers because of its stability.

5. Arch Linux

Arch Linux targets only advanced users. Basically arch CD has nothing except the base system in it. All you can see is just a blank screen with cursor blinking for you to enter commands and build your operating system. Arch is a bleeding edge Linux distribution, that means you just have to install it once and update whenever there is a new release and new versions of software’s are immediately available waiting for you to install from their repository. It follows the principle of “KISS” ‘keep it simple, stupid’. You will have complete control over what has to be installed into your system from the basement. In a word you can say that we are customizing operating system according to our need.

Lets have our 2nd byte – Essential Software’s for Web Development

Essential Software’s

As a web developer you have some specific software’s to work with and achieve you goals. You may be familiar with all the windows software’s that you were using, Well some of the known software’s are also used in Linux. I am gonna list out them first before we go any further.

To set-up local Web Server

  • Apache Server
  • PHP
  • MySql Server
  • PhpMyAdmin (optional)

To setup web development tools (Best according to me)

  • Graphic Designing Tools – GIMP(Photoshop alternative), Inkscape(Illustrator alternative)
  • Code Editors – SublimeText2, Gedit, Kate etc..
  • FTP Clients – Filezilla

I guess the above tools are more than enough to get started with your basic needs to set-up a clean and simple Web Developing Environment in Linux

Now its time for our final byte – Installing to get started

Note: “I think installing one of the basic Linux Operating System wouldn’t be a hectic job so I am skipping this step and moving forward to the installation step. But If you need any guidance on installing any of the Linux operating system just make a request through a comment and i will get back to you in free time or post a separate article for this based on the number of request” Installing Software’s To Get Started

So for now let me assume that you have Linux system running (Either Ubuntu, Linux Mint or Fedora) on your machine. The following commands will only work for the Mentioned Operating Systems and might not work in any other Linux distribution if its not a derivative of the above.

First of all Open up a terminal window. Installing LAMP(Linux,Apache,Mysql,& PHP) Server

Ubuntu – If you are using Ubuntu or Linux Mint the following commands are to be executed. (Make sure internet is enabled to download these software’s)

This command will install the Apache server for you

$sudo apt-get install -y apache2

This command will install the PHP for you

$sudo apt-get install -y php5

This command will install the PhpMyadmin for you (optional)

$sudo apt-get install -y phpmyadmin

This command will install the MySQL-server. You will be prompted to enter your MySQL password. Enter the password twice and remember it, we need this to test the mysql server.

$sudo apt-get install -y mysql-server

Installing some Developement software’s

This command will install the GIMP software.

$sudo apt-get install -y gimp

This command will install the Inkscape software.

$sudo apt-get install -y inkscape

This command will install the ftp client software.

$sudo apt-get install -y filezilla

This command will install all the development bundles required for building software’s from source-code. Required for SublimeText2 Editor

$sudo apt-get install -y build-essential

Short command to execute all these

$sudo apt-get install -y apache2 php5 mysql-server phpmyadmin gimp inkscape filezilla build-essential

Kate is default text editor for KDE Desktop Environment and can be found in KUBUNTU (UBUNTU distribution with KDE Desktop Enviroment instead of Unity)

Gedit is default text editor for Ubuntu, Linux Mint and Fedora

SublimeText2 Editor is similar to Mate-Text-Editor of Macintosh. Since this is not available in the official software repository of any operating system we need to download it from their site and install.

After downloading extract the bzip2 file and make the “sublime_text” file executable.

You can also extract the bzip2 file from terminal with the following command

Go to the download folder

$cd Downloads/

Extracting command

$tar -jxvf sublimetext.tar.bzip2

After extracting there is a new folder created in Downloads folder

$cd sublimetext

You can use the ls command to list the files in the folder

$ls

Make the sublime_text file executable

$chmod +x sublime_text

Now you can double click on sublime_text file to start the program, Make sure build-essential tools are installed for the program to start

With this we are ready go forward and work on web development projects in Ubuntu.

Fedora – If you are using fedora then execute the following commands. ‘.i686 is used if you are using 32bit version and .x86_64 is used if you are using 64bit version of the Fedora Operating System’

This command will install Apache Web Server

$sudo yum install httpd.i686

This command will install MySQL-server - You will prompted to enter your MySQL password. Enter it and remember we need this later to test the MySQL server.

$sudo yum install mysql-server.i686

This command will install latest version of PHP

$sudo yum install php.i686

This will install latest version of gimp

$sudo yum install gimp.i686

This will install latest version of Inkscape

$sudo yum install inkscape.i686

This will install latest version of Filezilla

$sudo yum install filezilla.i686

This will install Development Tools for fedora This is an alternative of “build-essential” for Fedora

$sudo yum groupinstall -y "Development Tools"

Short command to execute all these, except Development Tools

$sudo yum install httpd.i686 mysql-server.i686 php.i686 gimp.i686 inkscape.i686 filezilla.i686

So now that we have installed all the necessary software’s lets proceed and test whether the server is running perfectly. Ubuntu

To start the apache web server

$sudo service apache2 start

To start the mysql server

$sudo service mysql start

Fedora

To start the apache server

$sudo service httpd start

To start the mysql server

$sudo service mysqld start

After starting the apache server point you browser address to http://localhost or http://127.0.0.1 You should get a page displaying..

Ubuntu – “It works!” Fedora – “You get a simple web page displaying logos of Apache and Fedora”

Now that our web server is running lets test the mysql server

Open up a terminal and follow the commands.

Ubuntu or Fedora

$mysql -u root -p

Enter your password ****** and you should be presented with a mysql command promt like

mysql>

If you are able to login successfully then its confirmed that your MySQL server is ready for creating new databases
