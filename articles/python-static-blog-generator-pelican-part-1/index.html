What is Pelican and why Pelican?

Pelican is a python based static blog generator. In simple words we can say that an application that uses simple python commands to generate html blogs. You don’t have to worry too much about modifying contents on each html pages for a simple modification. Because pelican will take care of it to repeat multiple times. The well known competitor for Pelican is Nikola. Nikola is another python based static blog generator. Pelican has a huge community and contributors for the project than Nikola, and this is the main reason why I chose to write tutorial on Pelican over Nikola.

Even this site bitben.org is built using pelican with a custom pelican theme.

How to install pelican on Linux?

Installing pelican is very simple in Linux. Follow these set of commands to install pelican on Linux.

#one simple command to install pelican
$sudo pip install pelican

#Installing pelican on a virtual python environment is recommended because your whole project including the python libraries and packages can be kept in a single folder and it will be easy to backup on different stages.

#Install virtual environment package
$sudo apt-get install virtualenv 	#for ubuntu users
$su -c "yum install virtualenv"    	#for fedora users

#create a virtual environment in a folder called pelican-blog
$mkdir pelican-blog
$cd pelican-blog
$virtualenv package-python
#This will install virtual python package in the folder package-python

#lets activate the new python virtual environment 
$. package-python/bin/activate

#This will install pelican under virtual environment. Now you are able to run pelican commands only under the virtual environment and not globally.
$pip install pelican

#by default pelican uses ReST (restructured text) syntax for writing articles. But if you are a fan of MarkDown syntax, then you may have to install markdown as well.
$pip install markdown

Now python pelican has been successfully installed and ready to create a new project for generating static blogs.

With this you have created a project directory with necessary configuration files to build your html blog.

Generate pelican blog

$cd pelican-blog
#go to pelican-blog directory

$make html
#generate the html blog

$cd output
#goto the generated blog directory

$python -m SimpleHTTPServer
#start a simple http server. Now you can view your blog at http://localhost:8000

With this now you are able to see a simple blog with default theme running at http://localhost:8000.
