Awesome 101 vim commands

In my previous article I had spoken about the powerfull VIM editor for web developers Powerful Vim Editor For Web Developers . I am here by writing this article with 101 vim commands that comes handy during your projectâ€™s development stage. Each of these codes were tested very well before posting it here and current version of VIM is 7.4 as of this article was posted. Comments section will be open for your queries and feedback.

$ vim file.txt          #open a text file in vim
$ i                     #insert mode on
$ esc-key               #insert mode off
$ u                     #undo
$ ctrl+r                #redo
$ :44                   #goto line 44
$ v                     #visual mode for copy and paste.
$ y                     #copy
$ d                     #cut
$ x                     #delete
$ p                     #paste
$ vim -o file1 file2    #open multiple files at once
$ a                     #append after the cursor
$ i                     #insert before the cursor
$ o                     #open line below
$ O                     #open line above
$ :r filename           #insert contents of file
$ cw                    #change word
$ cc                    #change contents in a line
$ c$                    #change to the end of line
$ rc                    #replace character with c
$ R                     #replace typeover
$ s                     #substitute a character
$ S                     #substitute a line
$ .                     #repeat last change
$ ctrl+h                #back one character
$ ctrl+left             #back one word
$ ctrl+u                #back to the begining of insert
$ ctrl+right            #forward to one word
$ :w                    #write changes to the file
$ :wq                   #write changes to the file and quit
$ :q!                   #quit without changes
$ zz                    #same as :wq
$ sh                    #execute shell commands
$ ctrl+d                #scroll down
$ ctrl+u                #scroll up
$ ctrl+f                #page forward
$ ctrl+b                #page backward
$ /string               #search forward
$ ?string               #search backward
$ ctrl+g                #display current line number and file info
$ n                     #repeat search
$ N                     #repeat search reverse order
$ G                     #Goto last line
$ nG                    #Goto to line n
$ :n                    #Goto to line n
$ H                     #move cursor to upper left corner
$ M                     #move cursor to middle of the line
$ L                     #move cursor to lower left corner
$ h                     #back a character
$ j                     #down a line
$ k                     #up a line
$ ^                     #begining of the line
$ $                     #end of the line
$ l                     #forward a character
$ w                     #forward a word
$ b                     #backward a word
$ fc                    #find c
$ ;                     #repeat find
$ dd                    #delete lines
$ dw                    #delete word
$ d)                    #delete to the end of sentence
$ D                     #delete to the end of line
$ x                     #delete character
$ u                     #undo a last change
$ U                     #undo all changes on line
$ :set list             #show invisible characters
$ :set nolist           #dont show invisible characters
$ :set number           #show line numbers
$ :set nonumber         #hide line numbers
$ :set autoindent       #indent after hitting enter
$ :set noautoindent     #turn of autoindent
$ :set showmatch        #show matching sets of paranthesis as they are typed
$ :set noshowmatch      #turn of showmatch
$ :set all              #show values of all possible parameters
