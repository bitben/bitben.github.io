Powerful commandline tools for web developers

posted on: 09-08-2013 10:10 AM / written by: bensonshaji / For: developers

Popularity

Tweet

LINK CHECKER

This is a tool used to check the broken links of a website or web application. This utility is written purely in Perl and is available for experimenting via website through this URL W3C Link Checker. There is also a command line utility so that it can be directly used via Linux shell during the development stage, installing procedures are explained in detail here W3C link checker – command line

Install linkchecker in Ubuntu/LinuxMint/Debian

$sudo apt-get install linkchecker

Install linkchecker in Fedora

$su -c "yum install linkchecker"

DOM

DOM is a command line utility to check the domain name availability in any nix system. Even though the availability of domain names can be checked via their website, this utility is very light weight and highlights the available and unavailable domain names with different colors. More than that performing an action in a command line is much fun :) . This tool is written in python using domai.nr ‘s json api and can be easily installed with PIP. More information on running dom is available through this URL DOM Github

Installing DOM in linux

$pip install dom

S3CMD AMAZON CLOUD

This is a command line tool for accessing simple storage service of amazon i,e AMAZON S3. This tool helps you to upload, retrieve and manage data in amazon S3. This tool is written in python and can be easily installed in any Linux system. More information on usage of s3cmd is available in their official site Amazon s3 tools

Installing s3cmd in Debian based systems

$sudo apt-get install s3cmd

Installing s3cmd in Fedora

$su -c "yum install s3cmd"

VOLO.JS

VOLO is one awesome tool which can speed up time consuming stages in any web development project. This is an automation tool with which you can quickly create projects, add libraries and automate the common task using node and JavaScript. Even though learning the commands of this tool is bit time consuming, in the end its worth your effort as it saves time and energy while working on your project. More information on this utility is available at github Github volo. If you find any difficulty in installing volo or nodejs kindly contact me for help.

Installing volo in linux

$npm install volo

VIM

Vim is one of the most powerful tool which every web developer need to learn. Because this is the tool that come in handy when you don’t have time to debug the code in your favorite fancy code editor. VIM is the default code editor in most of the Linux web hosting platform that gives you a feature called SSH. It is a command line utility for editing codes in the Linux shell. You need to learn few vim commands in order to start using it. Once you have learned using it properly then you may not feel the need of fancy code editors. Installing vim is very simple in all Linux distros.

Installing VIM on Debian based systems

$sudo apt-get install vim

Installing VIM on fedora

$su -c "yum install vim"

CSSTIDY, HTML Tidy and JShint

Clean, tidy and robust are the essential part of any code. you need to keep it as simple as possible in the production servers because when your codes are lighter, it reduces the loading time of your website. Resulting increased performance in terms of speed. Server side languages are not considered to be small in size because it doesn’t have to be loaded on to the browser. languages like HTML, CSS and JavaScript are the ones which is responsible for the performance of a website. So these things have to be optimized very well. “csstidy” “tidy” and “JS hint” are the tools that comes handy for optimizing “CSS” “HTML” and “JavaScript” respectively. These tools can be installed easily in Linux via following commands.

Installing CSSTIDY

$sudo apt-get install csstidy 	
$su -c "yum install csstidy"

Installing HTML Tidy

$sudo apt-get install tidy		
$su -c "yum install tidy"

Installing JS hint

$npm install jshint

cURL

Curl is a command line utility used for multi purpose, basically it is defined as the tool that can transfer data using various protocols such as DICT, FILE, FTP, FTPS, HTTP, HTTPS, IMAP, IMAPS, LDAP, LDAPS, POP3, POP3S, RTMP, RTMPS, RTSP, SCP, SFTP, SMTP, SMTPS, Telnet, Gopher, TFTP and much more. Curl supports SSL certificates and HTTP PUT or HTTP POST features too. In a simple statement we can say that curl can get information and send information via one of the protocol mentioned above. How is it useful for web developers? Curl can send response headers of any website, it can be used to find out the look-up time, connect time or generally saying to check the time consumed in order to reach a website. Its uses are unlimited and it comes as a default software in most of the Linux operating systems. In case if it is not available, you can install it easily with this command.

Install curl in Linux

$sudo apt-get install curl 
$su -c "yum install curl"

RSYNC

rsync is a command line utility used to synchronize files from one location to another by minimize the data transfer using delta encoding when needed. This tool is very useful for web developers to synchronize their project in different places, either it a local drive or an online server. This tools helps in minimizing the data transfer for those limited bandwidth users. hence rsync is also considered as one of the essential tools for web developers. Installing rsync is very simple

$sudo apt-get install rsync 
$su -c "yum install rsync"

GIT

Git is one amazing tool for web developers, web designers or anyone who wants to maintain different versions of their project. I also have written an article about git. I hope you might like it. More information about GIT can be read from here Git for web developers.

SSH & SCP

I hope you already know about SSH and SCP. These are the most common tools used to login to your hosting account/server and transfer files securely. These are similar to ftp but instead of ftp commands you can use most of the Linux command features provided by your web hosting server.

If you know any other tools which can make a huge difference in a web developers daily life, kindly mention in your comments so that this article can be updated if it is relevant to the topic.
