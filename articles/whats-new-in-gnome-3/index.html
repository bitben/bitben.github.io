Whats new in gnome 3.10

posted on: 26-09-2013 21:27 PM / written by: bensonshaji / For: linux

Popularity

Tweet

Gnome has recently announced its latest version of desktop environment 3.10. As usual this release includes lot of new applications, improved GUI and lot other improvements than its previous version 3.8. Wanna test it? Quickly download Fedora 20 Alpha and start testing these new features. You can hope the stable version of fedora will be released in a couple of months.

New features that is packed in Gnome 3.10

Wayland display servers

The latest version of Gnome has replaced X servers with Wayland, a standalone display server; As they found it to be easier to develop and maintain. Gnome has promised to deliver smoother graphics, with improved animation and transitions. This technology is intended to provide greater flexibility for developers.

Integrated system status

In the previous version of Gnome we could see that the system status menu was made of series of buttons that was arranged on the top right corner of the desktop, this has been changed into a single menu button that is a combination of all menu displayed in a drop down list. This also includes a new screen brightness slider, battery status and rounded buttons featuring power off, lock and settings.

New header bars

Latest version of Gnome is introduced with combined header bar (title bars + menu bars), giving more space to the window content. This is one of the important feature we had been waiting for so long. In the previous versions, most of the space was being utilized for header content making the working area space less. Another issue was the close button, with the release 3.8 close button was removed when the window was maximized; making it difficult to click the gear button and then hitting the close. Thanks to gnome team for these improvements and modifications.

New Gnome Apps

Gnome has made a huge difference by introducing core application starting from 3.8. With release 3.10 we can see 5 major software introduction such as Gnome Maps, Gnome Music, Gnome SoftwareCenter, Notes and Gnome Photos. These apps looks cool but we expect more features added to these applications.
