Twitter via commandline

posted on: 09-08-2013 10:10 AM / written by: bensonshaji / For: linux

Popularity

Tweet

Once you have become a linux commandline freak, your mind tend to do most of the difficult things easily via commandline or linux shell. Consider me as an example. As I am a web designer and developer I would like to check the broken links of my website and this task can be accomplished by executing one simple command </pre> $linkchecker http://www.bitben.org </pre>

Simple and clean, this list all the broken links from my website http://bitben.org. If I am not a commandline freak then i would do the task by visiting google and search for linkchecker website of W3C link checker. Then entering my website url in the input form and waiting for the result to be shown. Here the concepts are the same but are executed in 2 different ways. Even though i mentioned that the concept is same, the time required for this purpose is definitely not the same, commandline execution is much much faster than the web browser process.

This is one of the main reason for me to explore web for commadline tools which makes my world simple and fast. “T” is one of those powerful commandline tools used for twitter tweets and much more you can do in a twitter website. This tool was developed in ruby and it can easily installed with one command.

Install T
$gem install t
Send a tweet
$t update "This is a tweet via commandline using T - A ruby app"
Twitter timeline
$t stream timeline

These are just simple examples – Wide variety of examples are mentioned at the end of this article. sometimes you may end with head breaking errors while installing this tool. but it is really not a head breaking part. basic requirements before installing it are development-tools,ruby-devel . Beyond this if you are getting any errors kindly post a comment and I will help you to install it.

Awesome commands of T

Authorize your twitter account
$t authorize
List authorized accounts
$t accounts
Activate a twitter account outoff authorized multiple accounts
$t set active bensonshaji
Help
$t help
Update your status
$t update "This is a tweet via commandline"
Retrieve a twitter users information
$t whois @bensonshaji
List statistics of multiple users
$t stats @bensonshaji @twitter
Follow users
$t follow @bensonshaji
Follow multiple users
$t follow @bensonshaji @twitter @google
List all friends you follow
$t friends -l --sort=followers
Favorite the last ten tweets that mentions you
$t followings -l --sort=tweets | head -10 | awk '{print $1}' | xargs t unfollow
Show 100 latest tweets in your timeline
$t timeline -n 100
Stream your timeline
$t stream timeline
Search latest 20 tweets that matches the query linux
$t search all linux
Search tweets in your timeline that matches the query linux
$t search timeline linux
Convert any output of T to a spreadsheet .csv file
$t timeline --csv > output.csv

Note: T can backup upto 3200 tweets under any category such as tweets, retweets, favorites, mentions etc to a csv file. Backing up your tweets is explained in this blog post by JP, JP’S backing up tweets via T
